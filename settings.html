Settings

Profile

Click on the user profile in the top right corner and click “Settings” to navigate to the Settings page.

In the Profile section, you can manage your personal information and account settings. This includes the following functions:

  1. Update first and last name.

  2. Reset password.

Team

In the Team section, you can view your team name. Updating team name and adding members is coming soon. To update your team, please contact support@onyx-robotics.com.

API Keys

In the API Keys section, you can manage your API keys for accessing the Engine API. Your API key is used to authenticate requests to the Engine API. For security, your API key is only shown immediately after generation. If you regenerate your API key, your previous key will be invalidated. Be sure to copy and save your new API key in a secure place, as you will not be able to view it again after leaving this page.

  1. Click “Regenerate” to create a new API Key.

  2. Copy and save the API key in a secure place, as you will not be able to view it again after leaving the page.

License

In the License section, you can view your subscription plan. Contact us at support@onyx-robotics.com to discuss upgrading plans.
